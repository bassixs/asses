# Как пользоваться ботом ассессмент-центра

## Общая идея

Бот помогает провести ассессмент-центр по цепочке:

```text
ассессмент-центр -> участник -> упражнение -> аудио + блокнот -> заполненный блокнот -> отчет -> ИПР
```

## 1. Создать ассессмент-центр

```text
/create_center Резерв руководителей май 2026
```

Бот ответит:

```text
Создан ассессмент-центр #1
```

Запомните номер.

## 2. Добавить участника

```text
/add_participant 1 Иван Петров
```

Где `1` — ID ассессмент-центра.

Бот ответит:

```text
Создан участник #7
```

## 3. Добавить упражнение участника

```text
/add_exercise 7 Ролевая встреча
```

Где `7` — ID участника.

Бот ответит:

```text
Создано упражнение #12
```

## 4. Загрузить аудиозапись упражнения

Отправьте боту аудиофайл или голосовое сообщение.

Важно: Telegram Bot API отдает боту только файлы до лимита `TELEGRAM_DOWNLOAD_MAX_BYTES`
(по умолчанию около 20 МБ). Если запись больше, бот попросит уменьшить файл или
разделить интервью на части.

Бот расшифрует запись и вернет:

```text
Расшифровано. ID: 34
```

Где `34` — ID записи.

## 5. Привязать запись к упражнению

```text
/attach_record 12 34
```

Где:

- `12` — ID упражнения;
- `34` — ID записи.

## 6. Загрузить Excel-блокнот наблюдателя

Отправьте боту `.xlsx` файл блокнота.

Бот прочитает индикаторы и вернет:

```text
ID блокнота: 8
```

## 7. Привязать блокнот к упражнению

```text
/attach_notebook 12 8
```

Где:

- `12` — ID упражнения;
- `8` — ID блокнота.

## 8. Обработать упражнение

```text
/process_exercise 12
```

Бот:

1. Берет расшифровку упражнения.
2. Берет Excel-блокнот.
3. Отправляет индикаторы и транскрипт в YandexGPT.
4. Заполняет:
   - колонку D: `+`, `-` или `НЗ`;
   - колонку E: цитаты и таймкоды;
   - колонку F: уровень компетенции.
5. Возвращает заполненный Excel-файл.

## 9. Повторить для всех упражнений

Для каждого упражнения участника повторите шаги:

```text
/add_exercise
загрузить аудио
/attach_record
загрузить блокнот
/attach_notebook
/process_exercise
```

## 10. Сформировать отчет участника

Когда все упражнения обработаны:

```text
/generate_report 7
```

Где `7` — ID участника.

Бот сформирует DOCX-отчет по всем обработанным упражнениям:

- сильные стороны;
- зоны роста;
- рекомендации.

## 11. Сформировать ИПР

Сначала нужен отчет участника.

Потом:

```text
/generate_ipr 7
```

Бот сформирует DOCX-файл индивидуального плана развития по зонам роста.

## Полезные команды

Показать все ассессмент-центры:

```text
/centers
```

Показать участников центра:

```text
/participants 1
```

Показать упражнения участника:

```text
/exercises 7
```

Обычная оценка компетенций по одной записи:

```text
/assess 34
```

Заполнить блокнот без полной структуры ассессмент-центра:

```text
/fill_notebook 34 8
```

## Админка в Telegram

Открыть:

```text
/admin
```

Пароль:

```text
1172
```

В админке можно посмотреть статистику и удалить:

- записи;
- блокноты;
- оценки;
- заполненные файлы;
- все данные сразу.

## Важно

Сейчас итоговый отчет и ИПР формируются как `.docx` файлы в структуре, близкой к примерам заказчика.

PDF можно добавить следующим шагом через LibreOffice на сервере.
